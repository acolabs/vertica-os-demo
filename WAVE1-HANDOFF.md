## Key Design Decisions

1. **Default org is Smartsheet** (`org_smartsheet`) вЂ” stored in localStorage via OrgProvider
2. **All timestamps are Unix milliseconds** (Date.now() style) as specified
3. **Model is `claude-sonnet-4.5`** everywhere вЂ” no GPT references
4. **Audit log hash chain**: Genesis entry hashes "genesis", subsequent entries hash `prev_hash + action + resource_id + timestamp`
5. **KPI deployment inflection at day 30** (daysAgo(60)): Pre-deployment shows flat/poor metrics, post-deployment shows clear improvement curves with natural noise
6. **Removed pre-existing `/src/components/mc-reference/` and extra UI components** that had uninstalled dependencies (sonner, framer-motion)
7. **Port 3004** set in both dev and start scripts
