## 8. STEP-BY-STEP SOP: 30-DAY ASSESSMENT

### Step 1: Pre-Closing Preparation (Days -7 to 0)
1. **Request data package** from sell-side advisor
2. **Schedule Day 1 meetings** with all direct reports
3. **Prepare assessment templates** (see Section 10)
4. **Align with PE deal team** on key hypotheses to test

### Step 2: Week 1 - Data Immersion (Days 1-7)
1. **CRM Data Export**
   - Pull all opportunities (last 24 months)
   - Extract activity data (calls, emails, meetings)
   - Document data quality issues

2. **Financial Reconciliation**
   - Validate ARR calculation methodology
   - Reconcile bookings vs. billings vs. revenue
   - Identify revenue recognition issues

3. **Org Assessment**
   - Request employee census
   - Map reporting relationships
   - Identify span of control issues

### Step 3: Week 2 - Customer & Sales Immersion (Days 8-14)
1. **Customer Calls** (minimum 8)
   - Script: "What made you buy? What almost stopped you? What would make you buy more?"
   - Document: Decision criteria, buying process, competitive alternatives

2. **Sales Call Shadows** (minimum 10)
   - Observe: Discovery, demo, negotiation, closing techniques
   - Assess: Rep capability, process adherence, messaging

3. **Pipeline Review**
   - Stage all opportunities with fresh eyes
   - Identify stage confusion and definition gaps
   - Flag opportunities requiring immediate action

### Step 4: Week 3 - Talent & Technical Assessment (Days 15-21)
1. **Performance Calibration**
   - Request performance ratings for last 2 cycles
   - Identify top 20%, middle 60%, bottom 20%
   - Flag flight risks and retention priorities

2. **Compensation Analysis**
   - Document current plans
   - Benchmark against market (use Radford/Option Impact)
   - Identify misalignment issues

3. **Technical Review**
   - Product demo and roadmap review
   - Technical debt assessment
   - Scalability evaluation

### Step 5: Week 4 - Synthesis & Reporting (Days 22-30)
1. **Data Analysis**
   - Calculate baseline metrics
   - Identify trends and anomalies
   - Compare to benchmarks

2. **Priority Ranking**
   - List all identified issues
   - Score by impact and ease of implementation
   - Select top 5 priorities

3. **Report Creation**
   - Write 30-day assessment (see template)
   - Prepare board presentation
   - Define 60-day plan

## 9. FAILURE MODES & MITIGATION

| Failure Mode | Early Warning | Mitigation | Owner |
|--------------|---------------|------------|-------|
| **Management Resistance** | Missed meetings, delayed decisions | Executive coaching, board alignment | OP_A |
| **Data Quality Issues** | Inconsistent metrics, missing data | Data audit, system fixes | VP Ops |
| **Talent Flight** | Key resignations, low engagement | Retention bonuses, communication | CHRO |
| **Customer Churn Spike** | Health score decline, usage drop | Executive outreach, success plans | VP CS |
| **Pipeline Collapse** | >30% reduction post-reset | Conservative forecasting, quick wins | CRO_X |
| **System Implementation Failures** | Missed deadlines, user resistance | Phased rollout, change management | VP Ops |
| **Comp Plan Backlash** | Rep complaints, productivity drop | Communication, grandfathering | CRO_X + CHRO |
